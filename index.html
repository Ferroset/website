SysInfo

The Ferroserver infrastructure consists of two dedicated home lab systems, each optimized for specific tasks and running on reliable, enterprise-grade hardware repurposed for personal and creative projects.

Ferroserver 1 - Media & Gaming Hub

Built on an HP EliteDesk HTPC platform, this server leverages an Intel Core i5-6500T vPro processor with 16GB of RAM. Its primary role is managing a comprehensive Plex media library with up to 5TB of films and shows, ensuring smooth streaming across multiple devices. Additionally, it hosts a Minecraft server accessible through Playit, providing a stable gaming experience for friends and family.

Ferroserver 2 - AI & Processing Powerhouse

Running on a ThinkCentre HTPC with ZimaOS (the evolved successor to CasaOS), this system harnesses an AMD Ryzen 5 PRO 2400GE processor paired with integrated Vega 11 Graphics and 16GB of RAM. This server specializes in running custom-trained Large Language Models built exclusively on ethical AI foundations using public domain and FOSS datasets. It also handles video upscaling tasks through Handbrake, preprocessing content for the Plex library on Ferroserver 1.
